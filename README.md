# Data Analyst Portfolio

Đây là cấu trúc thư mục được tạo ra để chuẩn bị cho việc xây dựng trang web Portfolio cá nhân.

## Hướng dẫn nhập dữ liệu
Bạn hãy mở file `src/data/projects.js` và cập nhật thông tin thực tế của mình (Tên, Bio, Kỹ năng, và Danh sách các dự án).
Sau khi bạn đã điền đầy đủ dữ liệu và chuẩn bị xong ảnh ở mục `public/images`, Antigravity sẽ giúp bạn xây dựng và tối ưu toàn bộ giao diện một cách dễ dàng và đẹp mắt.

## Agent Skill liên quan
Bạn có thể yêu cầu: "Hãy sử dụng thông tin trong `projects.js` và skill `deploy-to-vercel` hoặc `web-design-guidelines` để xây dựng và tự động triển khai trang web của tôi".
